# WorkforceHub Client

Frontend da aplicação WorkforceHub - Sistema de registro de ponto (punch clock).

**Stack**: Vue 3 + TypeScript + Vite

## Configuração Rápida

### Dependências
```sh
npm install
```

### Desenvolvimento
```sh
npm run dev
```

### Produção
```sh
npm run build
```

### Lint
```sh
npm run lint
```
